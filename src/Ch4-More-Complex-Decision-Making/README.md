# Chapter 4 - More Complex Decision Making

## Exercise 

### [A] If `a = 10`, `b = 12`, `c = 0`, find the values of expression in the following table:

| Expression | Value |
| :--- | :--- | 
| `a != 6 && b > 5` | 1 |
| `a == 9 \|\| b < 3` | 0 |
| `!(a < 10)` | 1 |
| `!(a > 5 && c)` | 1 |
| `5 && c != 8 \|\| !c` | 1 |
